📄 DOCUMENTACIÓN DE API - FASTAPI

Esta documentación corresponde a los endpoints disponibles en una API construida con FastAPI.

🔐 AUTENTICACIÓN

🔷 GET /login

Descripción: Endpoint público para probar acceso.

{
  "message": "Te enseña el login"
}

🟩 POST /login

Descripción: Endpoint para autenticarse.

{
  "username": "string",
  "password": "string"
}

Respuesta:

{
  "message": "Te redirige a la página Index"
}

🌐 GENERAL

🔷 GET /index/

Descripción: Retorna mensaje de bienvenida.

{
  "message": "Bienvenido {Nombre de usuario},al Gestor de Tareas"
}

📅 TAREAS

🔷 GET /tareas

Descripción: Muestra todas las tareas registradas.

{
  "tareas": []
}

🟩 POST /tareas

Descripción: Crea una nueva tarea.

{
  "title": "string",
  "description": "string"
  "state": "string"
}

Respuesta:

{
  "message": "Tarea creada",
  "tarea": {
    "id": [id],
    "titulo": "string",
    "descripcion": "string"
  }
}

🟧 PUT /tareas/{tarea_id}

Descripción: Actualiza una tarea existente.

{
    "title": "string",
    "description": "string"
    "state": "string"
}

Respuesta:

{
  "message": "Tarea actualizada",
  "tarea": {
    "id": [id],
    "title": "string",
    "description": "string"
    "state": "string"
  }
}

🟥 DELETE /tareas/{tarea_id}

Descripción: Elimina una tarea por ID.

Respuesta:

{
  "message": "Tarea eliminada",
  "tarea": {
    "id": [id],
    "title": "string",
    "description": "string"
    "state": "string"
  }
}
