Guía Rápida de Uso - API de Tareas (FastAPI)

🔐 Acceso

Iniciar sesión

Ruta: /login

🧾 ¿Qué necesitas? Tu nombre de usuario y contraseña.

✅ Si todo está bien, te redirige a index.

📌 Funcionalidades disponibles

✅ Ver mensaje de bienvenida

Ruta: /index/

📋 Tareas

👀 Ver todas las tareas

Ruta: /tareas

🔎 Muestra la lista completa de tareas que has registrado.

➕ Crear una tarea

Ruta: /tareas/crear_tarea

✍️ ¿Qué debes enviar?

  • Título de la tarea
  • Descripción de lo que tienes que hacer
  • Estado:Pendiente,Hecho,En proceso

✅ Respuesta: Te confirma que la tarea fue creada.

✏️ Editar una tarea

Ruta: /tareas/{id}

🧾 ¿Qué necesitas?

  • El número de la tarea (ID)
  • El nuevo título y descripción

✅ Respuesta: Muestra que fue actualizada correctamente.

❌ Eliminar una tarea

Ruta: /tareas/{id}

🧾 ¿Qué necesitas?

  • El id de la tarea que deseas eliminar

✅ Respuesta: Muestra que la tarea fue eliminada.

ℹ️ Información adicional

- Todas las respuestas vienen en formato fácil de leer (JSON).

- Si te equivocas al escribir algo, la API te avisa.
